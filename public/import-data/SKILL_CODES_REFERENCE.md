
```json
{
  "skill_code": "M3.B",
  "overview_html": "<h2>Quadratic Formula</h2>...",
  "objectives": "Students will solve quadratic equations using the quadratic formula...",
  ...
}
```

## Notes

- **Total Skills:** 129 (42 English + 43 Math + 22 Reading + 22 Science)
- **Skill Code Format:** Subject prefix + cluster number + letter identifier
- **Case Sensitive:** Always use uppercase (e.g., `E1.A` not `e1.a`)
- **Exact Match Required:** Database lookups are case-sensitive and exact
- **No Code Column:** Skills use `id` as the primary identifier; `code` column is nullable/unused
